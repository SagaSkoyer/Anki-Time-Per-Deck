This addon fundamentally changes the user experience of using Anki away from " per day" towards setting "how many minutes do you want to spend on per day?" for any specific decks you choose.

In doing so, in my experience, it leads to a healthier relationship with Anki of setting budgeted time you wish to spend on SRS per day, and the system sets the reviews vs new card counts each day for you automatically based on those user-supplied scheduling variables in the background.
